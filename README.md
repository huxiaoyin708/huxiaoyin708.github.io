# Xiaoyin Hu

This repository contains the source for Xiaoyin Hu's personal homepage, built with Jekyll for GitHub Pages.

Common files:

- `_pages/about.md`: homepage content
- `_pages/publications.md`: publication list
- `_pages/experience.md`: education and experience
- `_config.yml`: site title, author profile, email, and sidebar settings

Local build:

```sh
bundle install
bundle exec jekyll serve
```
